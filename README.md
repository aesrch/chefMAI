# ChefMAI Backend (Ktor)

Prerequisites

Requirements:

1. **=Java Development Kit (JDK) 21**: We highly recommend [Eclipse Temurin JDK 21](https://adoptium.net/). 
   * *Windows Users:* Ensure you check "Set JAVA_HOME variable" during installation.
2. **MySQL Server**: You need a local instance of MySQL running on port `3306`.
3. **Git**: To clone the repository.

*(Note: You **do not** need to install Gradle manually. This project uses the Gradle Wrapper, which will download the correct version automatically).*

# Local Setup & Installation
